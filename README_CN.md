# Social-Seq 项目 🐭

语言🌐：中文🇨🇳 | [**English**](README.md)

欢迎来到 Social-Seq 项目！这是一个用于解析动物密切接触时的多动物分类、三维姿态重构、社交类别识别、社交序列标签。通过构造实时闭环光遗传干预系统，该项目有望用于治疗孤独症（ASD）社交障碍。✨
